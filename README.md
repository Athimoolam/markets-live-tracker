# Markets Live Tracker
![Markets Live Tracker Screenshot](screenshot.png)

A simple, local/offline-capable HTML dashboard to monitor:

- Global stock exchanges 
- Nordic markets (Sweden, Denmark, Finland, Norway, Iceland)
- Major crypto exchanges (always 24/7)

### Features
- Real-time open/closed/weekend/24-7 status
- Current local time in each market's timezone
- Clickable links to official exchange websites
- Compact side-by-side layout (optimized for desktop/large monitors)
- Nordic markets grouped with subheader
- Updates every 60 seconds

### How to use
1. Visit the live version: https://yourusername.github.io/markets-live-tracker/  
   (replace "yourusername" with your actual GitHub username)
2. Or download `index.html` and open it locally in any browser
3. Works offline after first load (timings are calculated locally)
